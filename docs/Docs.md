# Setup docs

## Chrome shortcuts

| Action          | Command                |
| --------------- | ---------------------- |
| New Tab         | **Ctrl + t**           |
| Close Tab       | **Ctrl + w**           |
| Next Tab        | **Ctrl + Tab**         |
| Previous Tab    | **Ctrl + Shift + Tab** |
| Next Page       | **Ctrl + ]**           |
| Previous Page   | **Ctrl + [**           |
| Reopen last Tab | **Ctrl + Shift + t**   |

## Aerospace bindings

| Action                      | Command                              |
| --------------------------- | ------------------------------------ |
| Move workspace              | **Alt + {ws-num}**                   |
| Move app to workspace       | **Alt + Shift + {ws-num}**           |
| Open terminal               | **Alt + Enter**                      |
| Close app                   | **Alt + Shift + Q**                  |
| Move to                     | **Alt + {h \| j \| k \| l}**         |
| Move app to                 | **Alt + Shift + {h \| j \| k \| l}** |
| Fullscreen                  | **Alt + f**                          |
| Layout vertical accordion   | **Alt + s**                          |
| Layout horizontal accordion | **Alt + w**                          |
| Layout tiles horizontal     | **Alt + e**                          |
| Reload config               | **Alt + Shift + c**                  |
| Resize mode                 | **Alt + r**                          |
| [On resize] Modify size     | **{h \| j \| k \| l}**               |
| [On resize] Exit            | **Esc \|\| Enter**                   |
